```python
# 支持的工具
- Ruff: 代码风格和语法检查
- MyPy: 类型检查
- Flake8: 代码质量检查

# 输出格式
- JSON: 结构化数据
- Markdown: 人类可读报告
```

### **智能分类算法**
```python
# 分类策略
- import_errors: 导入相关问题
- syntax_errors: 语法错误
- type_errors: 类型错误
- style_issues: 代码风格问题
- security: 安全问题
- other: 其他问题

# 优先级排序
- critical → high → medium → low
- 文件分组: 按文件聚合问题
- 时间估算: 基于问题数量的智能估算
```

### **任务生成策略**
```python
# 任务设计原则
1. 独立可执行: 每个任务都可以独立完成
2. 时间可控: 15-90分钟，符合1-2小时粒度
3. 目标明确: 每个任务都有清晰的修复目标
4. 验证简单: 完成后可以立即验证效果
```
