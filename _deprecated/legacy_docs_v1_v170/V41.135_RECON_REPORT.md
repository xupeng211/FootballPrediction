## 🔍 关键发现

### 1. ✅ 真实 URL 格式突破（P0 - 核心成就）

**发现的 URL 格式**:
```
/football/germany/bundesliga-2023-2024/hoffenheim-bayern-munich-YeJyopM4/
/football/germany/bundesliga-2023-2024/wolfsburg-hoffenheim-jBEr38UF/
/football/germany/bundesliga-2023-2024/bayern-munich-wolfsburg-6NvEhyOD/
```

**URL 结构解析**:
| 部分 | 示例 | 说明 |
|------|------|------|
| 协议+域名 | `https://www.oddsportal.com` | 固定前缀 |
| 路径前缀 | `/football/` | 固定 |
| 国家 | `germany` | 联赛所在国家 |
| 联赛+赛季 | `bundesliga-2023-2024` | 联赛名称和赛季 |
| 队名+hash | `hoffenheim-bayern-munich-YeJyopM4` | **关键部分** |

**Hash 特征**:
- 长度: 6-8 个字符
- 格式: 大小写字母混合
- 示例: `YeJyopM4`, `jBEr38UF`, `6NvEhyOD`

**对比**:

| URL 类型 | 格式 | 结果 |
|----------|------|------|
| 旧格式（V41.134） | `/match/{id}/` | ❌ 空数据 |
| 新格式（V41.135） | `/football/{country}/{league}-{season}/{home}-{away}-{hash}/` | ✅ 页面可访问 |

### 2. ✅ V41.135 嗅探器升级成功

**核心改进**:

| 改进项 | V41.134 | V41.135 |
|--------|---------|---------|
| 页面等待模式 | `domcontentloaded` | `networkidle` |
| JS 渲染延迟 | 3 秒 | 5 秒 |
| 选择器策略 | 单一选择器 | 多选择器 + JavaScript 全面扫描 |
| Hash 检测 | 无 | 正则表达式检测大写字母 |

**关键代码**:
```python
# V41.135: 使用 networkidle
await page.goto(url, timeout=60000, wait_until="networkidle")

# V41.135: 深度等待
await page.wait_for_timeout(5000)

# V41.135: JavaScript 全面扫描
js_links = await page.evaluate("""
    () => {
        const links = [];
        const allElements = document.querySelectorAll('a, div[data-link]');

        allElements.forEach(el => {
            let href = el.getAttribute('href') || el.getAttribute('data-link');

            // Hash 特征: 包含大写字母或长字符串
            if (href && href.includes('/football/') &&
                (href.length > 5 || /[A-Z]/.test(href))) {
                links.push({href: href, text: el.textContent});
            }
        });

        return links;
    }
""")
```
