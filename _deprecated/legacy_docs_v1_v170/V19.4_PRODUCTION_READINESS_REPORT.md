## 5. Docker 生产部署验证

### 5.1 一键部署命令

```bash
# 1. 构建镜像
docker build -t footballprediction:v19.4 .

# 2. 启动服务栈
docker-compose up -d

# 3. 健康检查
docker-compose exec app curl http://localhost:8000/health/quick

# 4. 查看日志
docker-compose logs -f app
```

### 5.2 环境变量配置清单

| 环境变量 | 是否必需 | 默认值 | 说明 |
|----------|----------|--------|------|
| `ENVIRONMENT` | 否 | production | 运行环境 |
| `DOCKER_ENV` | 否 | false | Docker 环境标识 |
| `DB_HOST` | 否 | localhost | 数据库主机 (Docker: db) |
| `DB_NAME` | 否 | football_db | 数据库名称 |
| `DB_USER` | 否 | football_user | 数据库用户 |
| `DB_PASSWORD` | **是** | - | 数据库密码 |
| `REDIS_HOST` | 否 | localhost | Redis 主机 (Docker: redis) |
| `SECRET_KEY` | **是** | - | 应用密钥 (>=32字符) |
| `FOTMOB_X_MAS_HEADER` | 否 | - | FotMob API 请求头 |
| `SCHEDULER_ENABLED` | 否 | true | 定时任务开关 |
| `RISK_CONTROL_ENABLED` | 否 | true | 风控开关 |
