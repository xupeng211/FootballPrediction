## ⚠️ 技术挑战：实时捕获失败

### 问题分析

**现象**：Playwright 的响应处理器无法检测到 OddsPortal 的 `ajax-sport-country-tournament-archive_` API 调用

**已尝试的解决方案**：
1. ✅ `wait_until="networkidle"` - 等待网络空闲
2. ✅ 深度滚动触发（15 次 `mouse.wheel(0, 800)`）
3. ✅ 增加重试机制（3 次重试）
4. ✅ 延长等待时间（60 秒超时）

**可能根本原因**：
1. OddsPortal 使用 WebSocket 而非 HTTP 请求
2. API 调用需要特定的用户交互序列
3. 响应处理器注册时机问题
4. Service Worker 拦截了请求
