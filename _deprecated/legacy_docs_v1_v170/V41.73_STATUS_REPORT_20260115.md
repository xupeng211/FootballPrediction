## 根本原因分析

### 网络连接问题

1. **代理配置问题**:
   - 19 端口代理 (7891-7909) 无法连接到 oddsportal.com
   - `ERR_CONNECTION_CLOSED` 错误表明代理服务器未响应

2. **Playwright 浏览器问题**:
   - 即使网站可访问（curl 返回 200），Playwright 无法正确加载页面
   - 可能原因：
     - Shadow Ban（IP 被封禁）
     - 反爬检测触发
     - 页面加载超时

### 数据库连接问题

1. **本地 PostgreSQL 占用端口**:
   - WSL2 环境中本地 PostgreSQL 服务占用 5432 端口
   - 导致脚本连接到本地空库而非 Docker 数据库
