## ⚠️ 已知限制

### 1. Match ID 过期问题
- **原因**: FotMob API 对历史数据（2024年5月前）的 Match ID 不再返回有效数据
- **影响**: 无法使用过期 Match ID 进行采集测试
- **解决方案**:
  - 使用 FotMob API 的 leagueMatches 端点获取最新 Match ID
  - 或从数据库中获取最近采集的有效 Match ID

### 2. FotMob API 404 响应
- **现象**: `/api/leagueMatches?leagueId=47` 返回 HTML 404 页面
- **原因**: API 端点格式可能已变更或需要特定 headers
- **建议**: 检查 `fotmob_x_mas_header` 和 `fotmob_x_foo_header` 配置
