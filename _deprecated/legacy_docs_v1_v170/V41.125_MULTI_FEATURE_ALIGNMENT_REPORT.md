## 🎯 多特征加权对齐引擎

### 加权策略

```
综合评分 = 队名相似度(50%) + 比分校验(30%) + 时间窗口(10%) + ID 映射(10%)

阈值: ≥ 0.85 允许数据库自动关联
```

#### 1. 队名相似度 (50%) - Levenshtein 距离

```python
# 使用 TeamNameNormalizer.fuzzy_match 计算相似度
home_sim = fuzzy_match(fotmob_home, oddsportal_home)  # 0-100
away_sim = fuzzy_match(fotmob_away, oddsportal_away)  # 0-100

# 归一化到 0.0-1.0，取平均，应用 50% 权重
avg_name_sim = ((home_sim + away_sim) / 2.0) / 100.0
name_score = avg_name_sim * 0.5  # 0.0 - 0.5
```

**关键修复 (V41.125)**:
- `fuzzy_match` 返回 0-100 范围（百分比）
- 必须除以 100.0 归一化到 0.0-1.0 范围
- 修复前评分显示 50.100，修复后正确显示 0.501

#### 2. 比分校验 (30%) - 标准化强校验

```python
# 标准化比分格式（"2:1" → "2-1"）
fm_score_normalized = _normalize_score(fotmob_score_str)  # 移除空格，统一分隔符
op_score_normalized = _normalize_score(oddsportal_score)

# 比分强校验：绝对相等
if fm_score_normalized == op_score_normalized:
    score_validation = 0.3  # 满分
else:
    score_validation = 0.0  # 比分不匹配，零分

# 无比分数据：不打分也不惩罚
if not fm_score_normalized or not op_score_normalized:
    score_validation = 0.0  # 跳过验证
```

**标准化规则**:
- 移除所有空格: `"2 - 1"` → `"2-1"`
- 统一分隔符: `"2:1"` → `"2-1"`
- 空字符串保持为空: `""` → `""`

#### 3. 时间窗口校验 (10%) - 线性衰减

```python
# V41.125 优化: 指数衰减 → 线性衰减（更宽松）
# 旧公式: exp(-diff/24) → 21h 时因子 0.42，过于激进
# 新公式: 1 - (diff/48) → 21h 时因子 0.56

if time_diff_hours <= 24:
    decay_factor = max(0.5, 1.0 - (time_diff_hours / 48.0))
    time_window_score = decay_factor * 0.1  # 0.05 - 0.10
else:
    time_window_score = 0.0  # 超过 24h，零分
```

**衰减曲线 (V41.125 优化后)**:
| 时间差 | 衰减因子 | 时间分数 |
|--------|----------|----------|
| 0h     | 1.00     | 0.100    |
| 12h    | 0.75     | 0.075    |
| 21h    | 0.56     | 0.056    |
| 24h    | 0.50     | 0.050    |

#### 4. ID 映射奖励 (10%) - alias_teams 历史

```python
if fotmob_home_id or fotmob_away_id:
    if _check_id_mapping_history(home_id, away_id):
        id_mapping_score = 0.1  # 有历史成功记录
    else:
        id_mapping_score = 0.05  # 有 ID 但无历史记录
else:
    id_mapping_score = 0.0  # 无 ID 数据
```

## 🛡️ 风险与限制

### 已知限制

1. **无比分数据**: 最高只能得 0.6 分，无法达到 0.85 阈值
   - **缓解**: 对于未完场比赛，延迟关联或人工复核

2. **队名相似度阈值**: 队名相似度 < 50% 时，即使其他维度完美匹配也无法通过
   - **缓解**: 使用 alias_teams 表维护已知队伍别名映射

3. **时间窗口**: 24 小时窗口可能错过延迟开球的比赛
   - **缓解**: 对于已完场比赛，可放宽到 48 小时

### 误报场景

| 场景 | 风险 | 缓解措施 |
|------|------|----------|
| 同一球队在短时间内有两场比赛 | 可能关联错误比赛 | 比分校验（30% 权重）有效拦截 |
| 队名拼写相似但不同球队 | 可能误关联 | 比分校验 + 时间窗口双重验证 |
| 延迟开球 > 24h | 可能被正确拦截 | 人工复核或放宽时间窗口 |
