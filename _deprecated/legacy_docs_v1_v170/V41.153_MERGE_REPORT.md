所有迁移后的类都采用了单元测试友好的设计：

```python
# 示例: LocalHtmlParser
class LocalHtmlParser:
    def __init__(
        self,
        config: Optional[VendorConfig] = None,
        debug: bool = False,
        price_pattern: Optional[re.Pattern] = None,
        selectors: Optional[List[str]] = None,
    ):
        # 依赖注入设计 - 便于 Mock
        self.config = config or VendorConfig()
        self.price_pattern = price_pattern or self.DEFAULT_PRICE_PATTERN

    def parse_file(self, file_path: Union[str, Path]) -> Optional[PriceExtractionResult]:
        # 纯函数设计 - 无副作用
        path = Path(file_path)
        html_content = path.read_text(encoding="utf-8")
        return self.parse_string(html_content)
```

**测试示例**:
```python
def test_local_html_parser():
    parser = LocalHtmlParser(
        config=VendorConfig(pinnacle="TestVendor"),
        debug=True,
    )
    result = parser.parse_string('<div>TestVendor 1.50 4.20 6.50</div>')
    assert result.source == "TestVendor"
    assert result.prices == [1.50, 4.20, 6.50]
```
