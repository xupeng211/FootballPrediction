## 🎯 V20.4.1 新增特性详解

### 1. Momentum 动量特征 (6维)
```python
# 提取源: momentum.main.data 数组
- momentum_mean: 平均动量值
- momentum_std: 动量标准差 (波动性)
- momentum_min: 最小动量值
- momentum_max: 最大动量值
- momentum_range: 动量极差 (max - min)
- momentum_positive_ratio: 正值分钟数占比
```

### 2. Bench Depth 替补深度 (9维)
```python
# 提取源: team.subs 数组
- home_bench_fw_count: 主队替补前锋人数
- home_bench_avg_rating: 主队替补平均评分
- home_bench_total_market_value: 主队替补总身价
- away_bench_fw_count: 客队替补前锋人数
- away_bench_avg_rating: 客队替补平均评分
- away_bench_total_market_value: 客队替补总身价
- diff_bench_fw_count: 替补前锋人数差
- diff_bench_avg_rating: 替补平均评分差
- diff_bench_total_market_value: 替补总身价差
```

### 3. Shot Quality 射门质量 (代码已实现)
```python
# 提取源: shotmap.shots 数组
- home_shot_distance_avg: 主队平均射门距离
- home_shot_distance_min: 主队最近射门距离
- home_shot_distance_max: 主队最远射门距离
- away_shot_distance_avg: 客队平均射门距离
- away_shot_distance_min: 客队最近射门距离
- away_shot_distance_max: 客队最远射门距离
```
**注**: 当前测试数据中 shotmap.distance 字段不可用，特征值为 0
