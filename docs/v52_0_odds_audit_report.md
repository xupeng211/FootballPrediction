## 三、负溢价比赛分析

### 3.1 定义

**负溢价 (Negative Margin)**: Margin < 1.0，意味着：
- 理论返还率 > 100%
- 庄家在每单上都会亏损
- **这在真实市场中不可能存在**

```
负溢价比赛: 478 场 (55.71%)
```

## 四、ROI 关联性分析

### 4.1 随机模型异常解释

**问题**: V51.8 随机模型平注 ROI 为何仅 -0.2%？

**答案**: 因为赔率数据缺乏庄家抽水！

```
正常市场 (Margin = 1.06):
  理论返还率 = 1/1.06 = 94.34%
  期望 ROI = 94.34% - 100% = -5.66%

当前数据 (Margin = 0.9887):
  理论返还率 = 1/0.9887 = 101.14%
  期望 ROI = 101.14% - 100% = +1.14%

实际随机模型 ROI = -0.2%
-> 与理论预期 +1.14% 接近 (差异来自样本波动)
```

**结论**: 赔率数据的数学结构与随机模型的 ROI 完全吻合，证明数据本身有问题。

### 4.2 模拟扣除 5% 庄家利润

|------|----------|------|
| **原始 V51.7** | **15.02%** | - |
| 扣除 5% 后 | **-4.22%** | **-19.24%** |

### 5.2 可能的数据来源

| 来源类型 | 可能性 | 证据 |
|----------|--------|------|
| **全市场最高赔率组合** | 🔴 **极可能** | 负溢价 55.71% |
| 合成/模拟数据 | 🔴 **可能** | 缺乏 bookmaker 标识 |
| 真实单一博彩公司 | ✅ **不可能** | 抽水率严重偏低 |

**推断**: 数据很可能是从多个博彩公司中提取各结果的最高赔率组合而成，这会导致：
- Home 胜赔率来自公司 A
- Draw 赔率来自公司 B
- Away 赔率来自公司 C
- 组合后的 Margin < 1.0

## 六、最终判决

### 6.1 赔率真实性判决

| 判决项 | 结论 |
|--------|------|
| **数据真实性** | 🔴 **非真实可交易赔率** |
| **数据可用性** | 🔴 **严重高估模型性能** |
| **实盘风险** | 🔴 **极高风险 - 不可直接使用** |

### 6.2 根本原因

**赔率数据是"全市场最高赔率组合"或"合成数据"，而非单一博彩公司的真实报价。**

这导致：
1. 抽水率严重偏低 (平均 0.9887)
2. 负溢价比例极高 (55.71%)
3. 回测 ROI 严重虚高

| V51.7 (2维) | 15.02% | ~-5% 至 0% | -100% |
| V51.9 (盲测) | 42.12% | ~5-10% | -75% |
