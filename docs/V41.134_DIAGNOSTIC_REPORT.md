**冒烟测试结果**:

| Match ID | 主队 | 客队 | URL 格式 | 赔率数据 |

| 4222004 | Bayer Leverkusen | Bayern München | `/match/{id}/` | ❌ 0 组 |
| 3919230 | Fiorentina | Torino | `/match/{id}/` | ❌ 0 组 |

**问题分析**:
- 当前使用格式：`https://www.oddsportal.com/match/{id}/`
- 实际需要格式：`https://www.oddsportal.com/football/{country}/{league}/{home}-{away}-{hash}/`
- **关键差异**：OddsPortal URL 需要独特的 hash 后缀（如 `ID0097YDEK`）
