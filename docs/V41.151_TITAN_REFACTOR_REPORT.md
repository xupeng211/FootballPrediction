### 3. 指数级退避重试引擎

**V41.150 问题**:
```python
# 无重试机制，一次失败直接跳过
async def capture_match_html(...):
    try:
        await page.goto(...)
    except Exception as e:
        logger.warning(f"捕获失败: {e}")
        return None
```

**V41.151 解决方案 - RetryEngine**:
```python
class RetryEngine:
    """指数级退避 + 抖动重试引擎"""

    def _calculate_delay(self, attempt: int) -> float:
        # 指数退避: delay = base_delay * (2 ^ attempt)
        delay = min(
            self.config.base_delay * (2.0 ** attempt),
            self.config.max_delay,
        )

        # 随机抖动: ±25% 防止惊群效应
        if self.config.jitter_enabled:
            jitter = delay * 0.25
            delay += random.uniform(-jitter, jitter)

        return delay

    def execute(self, func, operation_id, *args, **kwargs):
        for attempt in range(self.config.max_attempts):
            try:
                return func(*args, **kwargs)
            except Exception:
                if attempt < self.config.max_attempts - 1:
                    delay = self._calculate_delay(attempt)
                    time.sleep(delay)
        raise HarvestError("All attempts failed")
```

**重试时序示例**:
```
Attempt 1: 失败 → 等待 1.0s ± 0.25s
Attempt 2: 失败 → 等待 2.0s ± 0.5s
Attempt 3: 失败 → 抛出 HarvestError
```

**优势**:
- ✅ 指数级退避（避免服务器压力）
- ✅ 随机抖动（防止反爬识别）
- ✅ 冷却模式（连续失败自动暂停）
- ✅ 装饰器支持（`@with_retry`）

**V41.151 解决方案 - StructuredLogger**:
```python
class StructuredLogger:
    """上下文感知的结构化日志"""

    def _format_message(self, message: str, extra_context: Dict) -> str:
        ctx = {**self._context, **extra_context}
        if ctx:
            tags = " ".join([f"{k}={v}" for k, v in ctx.items()])
            return f"{message} | {tags}"
        return message

# 使用示例
```

### 5. 统计指标与报告系统

**V41.150 问题**:

**V41.151 解决方案 - MetricsCollector**:
```python
@dataclass
class OperationMetrics:
    """单个操作的指标数据"""
    operation_name: str
    total: int = 0
    success: int = 0
    failed: int = 0
    skipped: int = 0
    start_time: float = field(default_factory=time.time)
    end_time: Optional[float] = None
    average_quality_score: Optional[float] = None

    @property
    def duration(self) -> float:
        """计算耗时（秒）"""
        end = self.end_time or time.time()
        return end - self.start_time

    @property
    def success_rate(self) -> float:
        """计算成功率（百分比）"""
        if self.total == 0:
            return 0.0
        return (self.success / self.total) * 100

class MetricsCollector:
    """集中式指标收集器"""

    def generate_report(self) -> JSONType:
        """生成 JSON 格式统计报告"""
        return {
            "report_timestamp": datetime.utcnow().isoformat() + "Z",
            "summary": {
                "total_operations": len(self.operations),
                "total_items": sum(m.total for m in self.operations.values()),
                "total_success": sum(m.success for m in self.operations.values()),
                "overall_success_rate": ...,
            },
            "operations": {
                name: metrics.to_dict()
                for name, metrics in self.operations.items()
            },
        }
```
