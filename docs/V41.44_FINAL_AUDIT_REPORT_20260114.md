### 3. active_harvest() 全自动收割方法 (P0)

**核心功能**: 集成 Playwright + 隧道轮换 + 分页翻页

```python
async def active_harvest(
    self,
    league_name: str,
    max_pages: int = 10,
    headless: bool = True
) -> dict[str, int]:
    """V41.44: 全自动收割方法 - 集成 Playwright + 隧道轮换"""

    # 1. 获取联赛 URL（从 YAML 配置）
    league_config = self.LEAGUE_URLS.get(league_name)
    if not league_config:
        logger.error(f"❌ 联赛配置未找到: {league_name}")
        return {"total_harvested": 0, "total_updated": 0}

    # 2. V41.43: 启动前切换代理端口
    proxy_port = self.set_proxy_port()  # 随机选择 7890-7899
    logger.info(f"🔌 使用代理端口: {proxy_port}")

    async with async_playwright() as p:
        browser = await p.chromium.launch(
            headless=headless,
            args=[
                f"--proxy-server=http://127.0.0.1:{proxy_port}",
                "--disable-blink-features=AutomationControlled"
            ]
        )

        page = await browser.new_page()
        await page.goto(league_config["url"])

        # 3. 分页收割
        for page_num in range(1, max_pages + 1):
            logger.info(f"📖 正在处理第 {page_num} 页...")

            # V41.43: 每页切换代理端口（反爬策略）
            if page_num > 1:
                proxy_port = self.set_proxy_port()
                logger.info(f"🔌 切换代理端口: {proxy_port}")

            # 4. DOM 提取比赛信息
            matches = self.extract_matches_from_dom(page)

            # 5. UPSERT 入库
            for match in matches:
                self.upsert_match_hash(
                    fotmob_id=None,  # 从 matches 表查找
                    oddsportal_hash=match.hash_value,
                    oddsportal_url=match.url
                )

            # 6. 尝试翻页
            if page_num < max_pages:
                try:
                    await page.click(f"text={page_num + 1}")
                    await asyncio.sleep(5)  # 等待页面加载
                except:
                    logger.info(f"✅ 第 {page_num + 1} 页不可用，收割完成")
                    break

        await browser.close()

    return {
        "total_harvested": total_harvested,
        "total_updated": total_updated,
        "league": league_name
    }
```

**关键特性**:
- ✅ **隧道轮换**: 每页自动切换代理端口 (7890-7899)
- ✅ **DOM 直取**: 绕过 API 加密限制
- ✅ **分页翻页**: 自动点击下一页，最多 10 页
- ✅ **UPSERT 幂等**: 多次运行安全，不会重复插入
