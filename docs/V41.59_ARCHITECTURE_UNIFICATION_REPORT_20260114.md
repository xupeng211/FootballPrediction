### 2. 全异步化改造 ✅

**核心修改**: `src/services/hash_alignment_service.py`

**问题**: `RuntimeWarning: coroutine 'Page.evaluate' was never awaited`

**根本原因**: `extract_matches_from_dom()` 是同步函数，但调用了异步的 `page.evaluate()`

**解决方案**:
```python
# 修复前（V41.58）
def extract_matches_from_dom(self, page: Page) -> list[MatchInfo]:
    matches = page.evaluate(...)  # ❌ 同步调用异步方法

# 修复后（V41.59）
async def extract_matches_from_dom(self, page: Page) -> list[MatchInfo]:
    matches = await page.evaluate(...)  # ✅ 正确使用 await
```

**调用点修复**:
```python
# 修复前
matches = self.extract_matches_from_dom(page)

# 修复后
matches = await self.extract_matches_from_dom(page)
```

**RealDictCursor 兼容性修复**:
```python
# 修复前
with self.conn.cursor() as cursor:
    cursor.execute("SELECT current_database();")
    current_db = cursor.fetchone()[0]  # ❌ 索引访问

# 修复后
with self.conn.cursor(cursor_factory=psycopg2.extras.RealDictCursor) as cursor:
    cursor.execute("SELECT current_database();")
    result = cursor.fetchone()
    current_db = result['current_database'] if result else None  # ✅ 字典访问
```
