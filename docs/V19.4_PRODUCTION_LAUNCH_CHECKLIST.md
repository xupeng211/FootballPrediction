## 3. 风控系统状态 (Risk Control Status)

### 3.1 自动止损哨兵系统

#### 核心文件
| 模块 | 路径 | 功能 |
|-----|------|------|
| **Risk Monitor** | `src/ops/risk_monitor.py` | 自动止损哨兵 (5连亏 OR 15%回撤触发熔断) |
| **Price Verifier** | `src/ops/market_price_verifier.py` | 市场价格验证 (10%贬值阈值) |
| **Weekly P&L** | `src/ops/weekly_pnl_statement.py` | 每周实战对账单生成器 |

#### 风控阈值配置
| 阈值类型 | 默认值 | 说明 |
|---------|--------|------|
| `max_consecutive_losses` | 5 | 最大连续亏损次数 |
| `max_drawdown_pct_limit` | 0.15 (15%) | 最大回撤百分比 |
| `price_deprecation_threshold` | 0.10 (10%) | 价格贬值阈值 |

#### 风控系统测试
```bash
# 测试风控监控系统
python -c "
from src.ops.risk_monitor import RiskMonitor
monitor = RiskMonitor(initial_balance=1000.0)
summary = monitor.get_status_summary()
print(f'✅ Risk Level: {summary[\"risk_level\"]}')
print(f'✅ Balance: {summary[\"balance\"]}')
print(f'✅ Emergency Stop: {summary[\"is_stopped\"]}')
"
```

### 3.2 风控工作流集成
每日预测流水线 (`daily_workflow.py`) 已集成以下风控步骤：

1. **Stage 4 - 市场价格验证**: 在导出预测前对比当前价格与回测价格
2. **自动过滤贬值比赛**: 价格偏离 >10% 的比赛自动标记为 `[Price_Deprecated]`
3. **禁止导出贬值比赛**: 只有通过价格验证的比赛才会进入最终清单

### 3.3 每周对账自动化
```bash
# 生成每周对账单
python src/ops/weekly_pnl_statement.py
```

输出文件:
- `data/pnl/weekly_pnl_YYYYMMDD_to_YYYYMMDD.json` - JSON 格式报告
- `data/pnl/weekly_pnl_YYYYMMDD_to_YYYYMMDD.txt` - 文本格式报告
- `data/pnl/weekly_pnl_chart_YYYYMMDD_to_YYYYMMDD.png` - 可视化图表

## 8. 监控与告警 (Monitoring & Alerts)

### 8.1 日志监控
| 日志文件 | 路径 | 用途 |
|---------|------|------|
| 应用日志 | `logs/app.log` | 系统运行日志 |
| 每日工作流 | `logs/daily_workflow.log` | 每日预测流水线日志 |
| 错误日志 | `logs/error.log` | 错误追踪 |

### 8.2 风控告警阈值
| 告警类型 | 触发条件 | 响应 |
|---------|---------|------|
| **连续亏损** | >= 5 次 | 自动熔断 + 邮件告警 |
| **最大回撤** | >= 15% | 自动熔断 + 邮件告警 |
| **价格贬值** | >= 10% | 标记 `[Price_Deprecated]` + 跳过下注 |

### 8.3 每周报告
- **生成时间**: 每周一 00:00 UTC 自动生成
- **报告内容**:
  - 本周预测准确率 (整体 + 按结果类型)
  - 真实 Yield (扣除 5% 庄家抽水)
  - 混淆矩阵
  - 累计收益曲线
  - 每日盈亏分布
