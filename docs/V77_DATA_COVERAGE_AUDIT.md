## 💡 下一步建议

### 方案 A：批量 URL 构造（推荐）

放弃 HTML 提取哈希 ID，直接基于球队名称构造 URL：

```python
# 示例代码
def construct_url_directly(league: str, season: str,
                           home_team: str, away_team: str) -> str:
    # 使用 v75_url_builder.py 的标准化函数
    from scripts.v75_url_builder import normalize_team_name

    # 映射到 URL 路径
    league_slug = LEAGUE_MAPPING[league]  # premier-league
    country = COUNTRY_MAPPING[league]      # england

    home_norm = normalize_team_name(home_team)
    away_norm = normalize_team_name(away_team)

    url = f"https://www.oddsportal.com/football/{country}/{league_slug}-{season}/{home_norm}-{away_norm}/"

    return url
```

**优势**:
- ✅ 可批量处理 8,930 条缺失 URL
- ✅ 100% 名称匹配
- ✅ 不依赖 HTML 抓取

**风险**:
- ⚠️ 不含哈希 ID 的 URL 可能 404（需验证）
