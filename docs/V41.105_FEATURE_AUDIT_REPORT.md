```python

- home_ratio_xg              # 主队 xG 比例 (= home_xg / total_xg)

- home_ratio_shots_on_target # 射正比例
- away_ratio_shots_on_target
- home_ratio_passes_accurate # 传球成功率比例
- away_ratio_passes_accurate
- home_ratio_possession      # 控球率比例 (= home_possession / 100)
- away_ratio_possession
- home_ratio_aerial_duels_won # 空中对抗成功率

- ... (还有 26 个 ratio_*)
```

**问题**: 这些比例值可以通过原始特征计算得出，存在多重共线性
**建议**: ⚠️ 在 ML 模型中慎用，考虑降维 (PCA)
