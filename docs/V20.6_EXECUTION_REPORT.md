## 1. 核心算法代码展示

### 1.1 ShotmapAggregator 核心算法

```python
class ShotmapAggregator:
    """
    V20.6 射门数据聚合器 - 全量合成全息矩阵核心组件

    核心算法:
    1. 解析 shotmap.Periods[All/FirstHalf/SecondHalf] 数组
    2. 按 teamId 分类计算主客队指标
    3. 生成合成特征: home_FH_synth_xg, away_SH_synth_shots 等
    4. 提取动量特征: 峰值、波动率
    """

    PERIOD_MAP = {
        'All': 'all',
        'FirstHalf': 'FH',
        'SecondHalf': 'SH',
    }

    def aggregate_period(self, shots: List[Dict], team_id: int) -> ShotmapMetrics:
        """
        聚合单支球队在单个时段的射门数据

        核心计算:
        - total_shots: 射门事件总数
        - sum_expected_goals: xG 累加
        - avg_distance: 平均射门距离
        - shots_on_target_count: 射正次数
        - big_chance_count: 大机会次数 (xG >= 0.1)
        - peak_xg: 单次最高 xG
        - volatility: xG 标准差
        """
        metrics = ShotmapMetrics()
        team_shots = [s for s in shots if s.get('teamId') == team_id]

        if not team_shots:
            return metrics

        metrics.total_shots = len(team_shots)
        xg_values = []
        distances = []

        for shot in team_shots:
            # xG 累加
            xg = shot.get('expectedGoals', 0) or 0
            metrics.sum_expected_goals += xg
            if xg > 0:
                xg_values.append(xg)

            # 射正计数
            if shot.get('isOnTarget', False):
                metrics.shots_on_target_count += 1

            # 大机会计数 (xG >= 0.1)
            if xg >= 0.1:
                metrics.big_chance_count += 1

            # 距离计算 (基于球场坐标)
            x = shot.get('x', 0) or 0
            y = shot.get('y', 0) or 0
            distance = np.sqrt((100 - x)**2 + (34 - y)**2)
            distances.append(distance)

        # 平均距离
        if distances:
            metrics.avg_distance = np.mean(distances)

        # 动量指标
        if xg_values:
            metrics.peak_xg = max(xg_values)
            if len(xg_values) > 1:
                metrics.volatility = np.std(xg_values)

        return metrics
```

## 5. 数据流架构 (V20.6)

```
FotMob API
    ↓
L2 原始 JSON 存储
    ↓
┌─────────────────────────────────────────────────────────────┐
│              V20.6 特征提取引擎                              │
├─────────────────────────────────────────────────────────────┤
│ 1. stats.Periods [All/FirstHalf/SecondHalf]                 │
│    ↓ 如果缺失 FirstHalf/SecondHalf:                         │
│ 2. shotmap.Periods [All/FirstHalf/SecondHalf]               │
│    ↓ ShotmapAggregator 合成:                                │
│    - home_FH_synth_xg (合成半场 xG)                         │
│    - home_SH_synth_shots (合成半场射门)                      │
│    - 动量特征 (peak_xg, volatility)                         │
│ 3. MomentumExtractor 提取动量曲线特征                       │
│ 4. PlayerStatsExtractor 聚合球员明细                         │
└─────────────────────────────────────────────────────────────┘
    ↓
PostgreSQL 存储 (enriched_features JSONB)
    ↓
XGBoost 训练 (800+ 维现代赛季, 500+ 维历史赛季)
    ↓
生产模型部署
```
