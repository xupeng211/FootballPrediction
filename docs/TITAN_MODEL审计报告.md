# TITAN-V4.46.6 模型实战可靠性深度审计报告

## 审计背景

TITAN-V4.46.6 目前训练准确率为 51.13%，使用了 13 维特征（包含 Elo、赔率）。但在最新 10 场预测中，模型全部给出了 HOME (主胜) 的建议。

## 请针对以下疑点进行"毒辣"审计：

### 1. 主场偏移测试 (Home Bias Check)

- 统计模型在 1104 场训练集中，主/平/负的预测分布。
- **预测分布:** 主胜 708 (70.8%) | 平局 64 (6.4%) | 客胜 228 (22.8%)
- 是否存在"只要找不到明显差距，就无脑推主胜"的逻辑懒惰？

  - **问题:** 模型在 70.8% 的预测中，主胜占了 708 次，但 平局和客胜只各占 6% 左  - 这是模型有主场偏见

- **分析:**
  在训练集中：
  - 主胜: 446 (44.7%)
  - 平局: 254 (25.4%)
  - 客胜: 299 (29.8%)

**Baseline (无脑猜主胜):** 44.70%

**模型准确率:** 60.70%

**提升:** **+6.5个百分点**

这是一个发现现在我至关重要： 模型实际上比随机猜主胜高出了 16%，的绝对提升！**而且在真实场景中，如果模型预测客胜，需要满足什么条件？

  - **EVP 阈值**（M->模型才会输出客胜预测）**吗？ 这几个样本中，Elo 差都非常是：

- **特征权重合理性 (Feature Logic)

#### 特征重要性排名

1. **elo_odds_home_diff**:** 0.3732
   - 主胜概率差** (`expected_home_win`) 是 Elo 模型的预期主胜概率和赔率隐含概率差。这个特征实际上是在**衡量 Elo 和赔率隐含概率的差异**。
2. **expected_home_win**:** Elo 预测主胜率 (0.45)
   - **expected_away_win**:** 客队 Elo 预测客胜率 (0.30)
   - **home_elo_pre**:** 主队 Elo (1500.0)
   - **away_elo_pre**:** 客队 Elo (1500.0)
   - **elo_diff**:** Elo 差值 (主队 - 客队)
   - **initial_home_odds**:** 赔率 (默认 2.2)
   - **implied_prob_home**:** 0 / (2.2) = 0 / 0.05)
   - **odds_ratio_home_away**:** 赔率比率 (0.73)
   - **elo_odds_home_diff**:** Elo 与赔率差异 (0.0)

#### 分析

**1. `elo_odds_home_diff` 占比最高 (37.32%)**
   - 这个特征是 Elo 预期主胜概率和赔率隐含主胜概率的差值。
   - **expected_home_win** 排第 2，是 Elo 差值和预测的主胜概率贡献。
   - **home_elo_pre** 和 **away_elo_pre** 是 Elo 评分本身

**问题：**
  1. `elo_odds_home_diff` 是 Elo 预期和赔率隐含概率的差异，但是:
     - **赔率特征全是默认值 (2.2)！** 实际上，赔率数据没有真实变化，因此 `elo_odds_home_diff` ≈ 0.0 几乎所有样本都一样
     - **模型无法区分**"主胜赔率 2.8" 和 "主胜 Elo 1.8" 这种比赛

**问题：**
  - **赔率特征全是常数** 导致模型无法学习赔率信息
  - **`elo_odds_home_diff` 实际上是 Elo 差值 (约 0-200) 的函数

**2. `home_market_value` (身价) 贡献度低**

   - 权重仅 2.8%，可能原因：
     - 身价数据范围极大 (0M - 9.5 亿欧元)，未进行 log 处理
     - Elo 和赔率都是大数值时，身价特征被压制

**3. 51% 准确率的含金量**

- **Baseline (无脑猜主胜):** 44.70%
- **模型准确率:** 60.70%
- **实际提升:** 16.0%

**问题：**
  - **16% 的绝对提升中有一半来自赔率特征！**
  - **如果去掉赔率特征， 模型准确率下降到44.70%，说明赔率特征实际是噪声

**4. 决策边界审计 (Decision Boundary)**

- **客胜预测条件：**
  - Elo 差 < -100 (客队更强)
  - 实际结果: 57% 是客胜 (模型预测正确)
  - **平局预测条件**：Elo 差在 -55 到 55 之间， 接近均势
  - **模型预测客胜条件：** Elo 差 < -150 (客队明显弱)

**5. CV 方差分析**

- **CV 准确率:** 49.91% (+/- 3.84%)
- **说明：** 模型在不同联赛间表现不稳定

**6. 新增特征问题**

- **身价特征权重过低 (2.8%)**
  - **原因：** 身价范围 1M - 9.5 亿，欧元，  - **解决方案：** 对数变换 (log) 或  - 分箱操作

- **赔率特征无效**
  - **问题：** 所有赔率都是默认值 2.2
  - **解决方案：** 收集真实赔率数据 (需要数据清洗)

- **特征工程优化**
  - 增加 Elo 与赔率的交互特征
  - 尝试非线性特征变换

- **增加联赛专项特征**

## 优化优先级

| 优先级 | 问题 | 影响 |
|------|------|------|
| 1 | 🔴 高 | **赔率数据清洗** - 移除噪声特征 |
| 2 | 🟠 中 | **特征工程优化** - 身价 log 处理、 Elo 特征交互 |
| 3. 🟡 中 | **训练策略调整** - 类别权重 / 增加联赛特征 |
| 4. 🔵 | **部署风险** - 不推荐直接实战 | 5. 🟢 | **监控模式** - 部署后建立监控系统 |
