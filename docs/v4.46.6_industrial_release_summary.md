## 騡型使用策略

1. **主预测**: TITAN 模型 (优先)
2. **回退策略**: Elo + 赔率集成
3. **赛前预测**: 使用 Elo 预期进球作为代理特征

   - 计算 expected_home_goals 和 expected_away_goals
   - 计算特征向量
   - 调用 TITAN 模型预测
   - 如果模型不可用， 使用 Elo+赔率集成预测

4. **Kelly 准则**: 使用 Fractional Kelly (1/4 Kelly)
5. **数据库写入**: predictions 表 (UPSERT 模式)
6. **日志级别**: INFO
7. **并发支持**: 单线程预测
8. **缓存**: 模型元数据缓存
9. **试运行模式**: `--dry-run` (不写入数据库)
10. **输出格式**: 文本 (默认) / JSON (`--format json`)
11. **CLI 参数**:
    - `--limit N`: 最大预测数量 (默认: 50)
    - `--league NAME`: 按联赛过滤
    - `--dry-run`: 试运行模式
    - `--format text/json`: 输出格式
    - `--model-version`: 模型版本标识 (默认: TITAN-V4.46.6)
    - `--verbose`: 详细输出 (默认: False)
