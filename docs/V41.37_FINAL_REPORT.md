## 💡 根本原因分析

### OddsPortal 数据源的三重限制

1. **日期路由不可靠**
   - `#/date/20230918/` 返回 2024-05-19 的比赛
   - 跨度可达 8 个月

2. **搜索功能不完整**
   - 搜索结果页只显示球队页面
   - 不包含比赛详情链接

3. **重定向机制失效**
   - Hash Hunting 点击后跳转到错误页面
   - 无法通过队名搜索找到比赛

### 假设：OddsPortal 网站架构变化

可能的原因：
- OddsPortal 引入了新的客户端路由系统
- JavaScript 动态加载内容（SSR → CSR 迁移）
- 反爬虫机制加强（IP 限流、账号要求）
