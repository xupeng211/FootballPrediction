## 🛡️ 核心功能详解

### 1. 熔断器 (Circuit Breaker)

**设计目标**: 防止级联故障，连续失败达到阈值时自动熔断。

**状态机模型**:
```python
class CircuitState(Enum):
    CLOSED = "closed"      # 正常运行
    OPEN = "open"          # 熔断状态，拒绝请求
    HALF_OPEN = "half_open"  # 测试恢复

@dataclass
class CircuitBreakerConfig:
    max_consecutive_failures: int = 10  # 熔断阈值
    timeout_seconds: float = 300.0      # 冷却时间
    half_open_max_calls: int = 3        # 恢复测试次数
```

**状态转换图**:
```
CLOSED ─[连续失败≥10次]→ OPEN
OPEN ─[5分钟冷却]→ HALF_OPEN
HALF_OPEN ─[3次成功]→ CLOSED
HALF_OPEN ─[任何失败]→ OPEN
```

**使用示例**:
```python
# 初始化熔断器
cb_config = CircuitBreakerConfig(max_consecutive_failures=10)
circuit_breaker = CircuitBreaker(cb_config)

# 检查熔断器状态
if not circuit_breaker.is_closed():
    raise CircuitBreakerOpenError("Circuit breaker is OPEN")

# 记录成功/失败
circuit_breaker.record_success()  # 重置失败计数
circuit_breaker.record_failure()  # 增加失败计数
```

**熔断器状态信息**:
```json
{
  "state": "closed",
  "failure_count": 3,
  "max_consecutive_failures": 10,
  "last_failure_time": "2026-01-17T16:00:00",
  "trip_time": null
}
```

### 2. 预检探针 (Pre-flight Checks)

**检查项目**:

| 检查项 | 阈值 | 失败后果 |
|--------|------|----------|
| 磁盘空间 | ≥ 1.0 GB | ❌ 拒绝启动 |
| 代理延迟 | ≤ 5000ms | ⚠️ 警告/继续 |
| 数据库连接 | 5 秒超时 | ❌ 拒绝启动 |
| 目录结构 | 存在性 | ⚠️ 警告/自动创建 |

**实现代码**:
```python
class PreFlightChecker:
    MIN_DISK_GB: float = 1.0
    MAX_PROXY_LATENCY_MS: float = 5000.0
    DB_TIMEOUT_SECONDS: float = 5.0

    def run_all_checks(self) -> Tuple[bool, List[PreFlightCheck]]:
        checks = [
            self._check_disk_space(),
            self._check_proxy_latency(),
            self._check_database_connection(),
            self._check_directory_structure(),
        ]
        all_passed = all(c.status != PreFlightCheckResult.FAIL for c in checks)
        return all_passed, checks
```

**检查结果输出**:
```bash

[PRE-FLIGHT] proxy_latency: Proxy latency OK: 234ms
[PRE-FLIGHT] database_connection: Database connection OK: 12ms
[PRE-FLIGHT] directory_structure: Directory structure OK
```

### 3. 敏感数据掩码 (Log Masking)

**自动脱敏模式**:

| 数据类型 | 正则模式 | 替换结果 |
|----------|----------|----------|
| URL 哈希 | `/([a-zA-Z0-9]{8,12})/` | `/[REDACTED]/` |
| 数据库密码 | `password="..."` | `password="***"` |
| API Token | `token="..."` | `token="***"` |
| 代理凭据 | `://user:pass@` | `://***:***@` |

**实现代码**:
```python
class LogMaskingFilter(logging.Filter):
    HASH_PATTERN = re.compile(r'/([a-zA-Z0-9]{8,12})/')
    PASSWORD_PATTERN = re.compile(r'(password|passwd|pwd)["\s:=]+["\s]*([^"\s,}]{8,})', re.IGNORECASE)
    TOKEN_PATTERN = re.compile(r'(token|api_key|secret)["\s:=]+["\s]*([^"\s,}]{20,})', re.IGNORECASE)
    PROXY_CRED_PATTERN = re.compile(r'://[^:]+:([^@]+)@')

    def filter(self, record: logging.LogRecord) -> bool:
        record.msg = self._mask_sensitive_data(str(record.msg))
        return True
```

**脱敏效果示例**:
```python
# 原始日志
"https://www.oddsportal.com/football/spain/laliga-2024/barcelona-madrid-4A6T7YOu/"

# 脱敏后
"https://www.oddsportal.com/football/spain/laliga-2024/barcelona-madrid/[REDACTED]/"
```

**掩码统计**:
