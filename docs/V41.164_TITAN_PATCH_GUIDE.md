## 🎯 三大核心组件

### 1. Mg30 隐身配置 (`src/core/scrapers/mg30_stealth.py`)

基于 Mg30 browser.js 经验移植的高级隐身内核，包含 6 大伪装模块：

| 模块 | 功能 |
|------|------|
| **Core** | Navigator 属性重写 (webdriver: false), Chrome Runtime 注入伪装 |
| **WebGL** | GPU 指纹深度伪装 (Intel/NVIDIA/AMD 随机切换) |
| **Canvas** | Canvas 指纹噪声注入 (±1px 微弱噪声) |
| **Audio** | Audio Context 指纹随机化 |
| **Font** | 字体指纹伪装 |
| **Timing** | Timing API 偏移 (±500ms) |

**使用方式**:
```python
from src.core.scrapers.mg30_stealth import get_combined_mg30_script

# 获取组合脚本
mg30_script = get_combined_mg30_script()

# 注入到 Playwright 页面
await page.evaluate(mg30_script)
```

### 2. Response JSON 拦截器 (`src/core/scrapers/response_interceptor.py`)

在 Playwright Response 层直接截获纯金 JSON 数据，无需 BeautifulSoup 解析 HTML。

**核心功能**:
- ✅ 拦截所有网络响应
- ✅ 自动识别 `match/1-1-` 等关键字
- ✅ JSON 数据验证和解析
- ✅ 自动保存到文件
- ✅ 支持自定义处理函数

**使用方式**:
```python
from src.core.scrapers.response_interceptor import ResponseInterceptor, InterceptorConfig
from playwright.async_api import async_playwright

async with async_playwright() as p:
    browser = await p.chromium.launch()
    page = await browser.new_page()

    # 创建拦截器
    config = InterceptorConfig(
        keywords=["match/1-1-", "pinnacle", "oddsportal"],
        save_all=True,
    )
    interceptor = ResponseInterceptor(config)
    await interceptor.attach_to_page(page)

    # 访问页面
    await page.goto("https://www.oddsportal.com/...")
    await asyncio.sleep(10)

    # 获取截获的数据
    jsons = interceptor.get_intercepted_jsons()
    print(f"截获了 {len(jsons)} 个 JSON")
```

    --action intercept-json \

## ⚙️ 配置选项

### InterceptorConfig

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `keywords` | list[str] | ["match/1-1-", ...] | 关键字列表 |
| `content_type_whitelist` | list[str] | ["application/json", ...] | Content-Type 白名单 |
| `url_whitelist` | list[str] | ["oddsportal.com", ...] | URL 白名单 |
| `save_all` | bool | False | 是否保存所有截获数据 |
| `output_dir` | str | "storage/intercepted_json" | 输出目录 |
| `max_intercepts` | int | 10000 | 最大截获数量 |
