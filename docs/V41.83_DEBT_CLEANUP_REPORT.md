### 具体改进

#### 1. 深度扁平化手术 ✅
**策略**: Guard Clauses (卫语句) + 提前返回

**重构前**:
```python
def get_healthy_proxy_port(self, excluded_ports=None):
    if excluded_ports is None:
        excluded_ports = set()
    
    if self.proxy_health_checker:
        self.available_ports = self.proxy_health_checker.get_available_ports()
        available = [p for p in self.available_ports if p not in excluded_ports]
    else:
        available = [p for p in self.proxy_ports if p not in excluded_ports]
    
    if not available:
        logger.warning("⚠️  没有可用的代理端口")
        return None
    
    return random.choice(available)
```

**重构后**:
```python
def get_healthy_proxy_port(
    self, excluded_ports: set[int] | None = None
) -> int | None:
    """V41.83: 获取健康的代理端口"""
    # V41.83: Guard Clause
    if excluded_ports is None:
        excluded_ports = set()

    # V41.83: 优先使用 ProxyHealthChecker 的可用端口
    if self.proxy_health_checker:
        self.available_ports = self.proxy_health_checker.get_available_ports()
        available = [p for p in self.available_ports if p not in excluded_ports]
    else:
        available = [p for p in self.proxy_ports if p not in excluded_ports]

    # V41.83: Guard Clause
    if not available:
        logger.warning("⚠️  没有可用的代理端口")
        return None

    return random.choice(available)
```

**改进**: 
- 添加类型提示
- 提前返回减少嵌套
- 完整文档字符串
