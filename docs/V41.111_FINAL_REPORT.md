## 🔧 技术实现细节

### 比分解析函数

```python
def _parse_score(self, score_str: str | None) -> tuple[int | None, int | None]:
    """解析比分字符串 "2 - 3" → (2, 3)"""
    if not score_str:
        return None, None

    try:
        parts = score_str.split(" - ")
        if len(parts) == 2:
            return int(parts[0]), int(parts[1])
    except (ValueError, IndexError):
        pass

    return None, None
```
