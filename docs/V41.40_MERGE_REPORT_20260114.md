## ✅ 已完成工作

### 1. Bug 审计发现

**问题**: 9,305 场数据被错误标记为 "Historical League / 2020-2025"

**根因分析**:
- FotMob 采集器缺少动态标签逻辑
- 未根据比赛时间戳划分真实联赛名
- 未精准分配赛季标签

**建议修复** (未在本次执行):
```python
# 在 FotMobCoreCollector 中添加动态标签逻辑
def infer_league_and_season(match_time: datetime, league_id: int) -> tuple:
    """根据比赛时间和 league_id 推断联赛名和赛季"""
    from src.api.collectors.fotmob_league_registry import FOTMOB_LEAGUE_REGISTRY

    league_info = FOTMOB_LEAGUE_REGISTRY.get(league_id)
    if not league_info:
        return ("Historical League", "2020-2025")

    # 根据比赛时间推断赛季
    year = match_time.year
    month = match_time.month

    if month >= 8:
        season = f"{year % 100}/{(year + 1) % 100}"
    else:
        season = f"{(year - 1) % 100}/{year % 100}"

    return (league_info.name, season)
```
