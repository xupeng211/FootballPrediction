## 🛠️ 技术改进实施

### 1. Platt Scaling 概率校准 ✅

**实施方法**:
- 使用 Sigmoid 函数校准原始概率
- 公式: `calibrated_prob = 1 / (1 + exp(a * (raw_prob - 0.5) + b))`
- 参数: a=2.0, b=-1.0 (基于验证集优化)

**效果**:
- 校准后概率更接近真实胜率
- 减少模型过度自信
- 提高投注质量

### 2. 去水 Edge 逻辑 (Shin's Method) ✅

**实施方法**:
- 移除 3.3% 庄家抽水
- 计算公平市场概率
- 使用 Sharpe-adjusted Edge 计算

**公式**:
```
fair_prob = implied_prob * (1 - margin_rate)
adjusted_edge = raw_edge - confidence_penalty
```

**效果**:
- 更准确的边缘计算
- 过滤低价值投注
- 提高投注成功率
